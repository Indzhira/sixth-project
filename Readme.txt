Проект: Обучение с учителем: качество модели

Необходимо разработать решение, которое позволит персонализировать предложения постоянным клиентам интернет-магазина, чтобы увеличить их покупательскую активность.

Постановка задачи:
1. Проанализировать текущую базу данных постоянных клиентов интернет-магазина
2. Собрать и систематизировать данные по четырём ключевым группам признаков: коммуникация с клиентом, продуктовое поведение покупателя, финансовое поведение клиента и поведение покупателя на сайте
3. Разработать модель, способную предсказывать вероятность снижения покупательской активности в ближайшие 3 месяца
4. Интегрировать дополнительные данные о прибыльности клиентов из финансового департамента
5. Используя данные модели и данные о прибыльности клиентов, нужно выделить сегменты покупателей и разработать для них персонализированные предложения

Для предсказания вероятности снижения покупательской активности в ближайшие три месяца были обучены модели:
1. KNeighborsClassifier
2. DecisionTreeClassifier
3. LogisticRegression

Лучшей моделью стала модель DecisionTreeClassifier, которая на тестовой выборке продемонстрировала высокое качество классификации с показателем ROC-AUC = 0.84.
